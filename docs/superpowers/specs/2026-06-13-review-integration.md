# 复盘方法论整合方案

## 两本书的核心精华

### 《复盘》沈磊 (2023) - 联想复盘法 3.0

**5大步骤：**
1. 梳理过程 - 还原事实，不带判断
2. 回顾目标 - 当初想做什么
3. 评估结果 - 实际做了什么，与目标的差距
4. 分析原因 - 为什么有差距，根因是什么
5. 总结经验 - 提炼规律，指导未来

**5大原则：**
- 开放心态 - 不自欺，接受事实
- 坦诚表达 - 说出真实想法
- 实事求是 - 用事实说话
- 反思自我 - 从自己身上找原因
- 集思广益 - 多角度思考

**复盘六字诀：**
- 慢 - 慢思考，发现细节
- 写 - 写下来，理顺过程
- 真 - 真实面对自己
- 问 - 自问自答
- 离 - 跳出当局者视角
- 行 - 落地执行

### 《复盘：对过去的事情做思维演练》陈中

**复盘的4个步骤：**
1. 回顾目标
2. 评估结果
3. 情景再现
4. 分析得失原因和总结规律

**复盘的关键：**
- 复盘 ≠ 总结，复盘的关键是**推演**
- 通过推演可以对各种可能性进行探讨
- 复盘是我们应对人生的本能，是刻意练习的内核

**复盘的层次：**
- 小事及时复盘
- 大事阶段性复盘
- 事后全面复盘

## 应用到项目的方案

### 当前项目的不足

现在的"每日反思"和"每周总结"只是简单的文字总结，缺乏结构化的复盘框架。用户记录了"做了什么"，但没有系统地思考"为什么"和"下次怎么做"。

### 整合方案

#### 1. 增强每日反思的结构

当前：AI 生成一段话总结

改为：结构化复盘模板

```
今日复盘 (2026-06-13)

【回顾目标】
今天计划做什么？

【实际完成】
实际做了什么？

【差距分析】
完成度如何？哪里有差距？

【原因分析】
为什么有差距？根因是什么？

【经验提炼】
下次遇到类似情况怎么做？
```

#### 2. 增加"推演"环节

在复盘中增加"如果...会怎样"的推演：

- 如果今天不做X，会怎样？
- 如果换一种方式做Y，结果会不同吗？
- 这个决策背后的假设是什么？假设成立吗？

#### 3. 周复盘增加深度

当前：简单的周总结

改为：结构化周复盘

```
本周复盘 (Week 24)

【目标回顾】
本周的核心目标是什么？

【结果评估】
- 完成了什么？
- 没完成什么？
- 有什么意外收获？

【关键事件分析】
选择本周最重要的1-2件事深度复盘：
- 事情经过
- 决策点
- 结果
- 如果重来会怎么做

【规律提炼】
本周发现了什么规律/模式？

【下周调整】
基于本周复盘，下周要调整什么？
```

#### 4. 增加"复盘标签"

让用户可以给复盘打标签，方便回顾：
- #成功经验
- #失败教训
- #认知升级
- #习惯养成
- #人际关系
- #项目管理

#### 5. AI 辅助提问

AI 不只生成总结，还要提出关键问题：
- "今天的工作中，哪个环节花了最多时间？为什么？"
- "今天有没有什么决策是凭直觉做的？背后的假设是什么？"
- "如果重来一次，你会怎么做？"

#### 6. 月度/季度复盘报告

自动生成成长报告：
- 本月复盘次数
- 常见的"坑"有哪些
- 成功模式有哪些
- 认知升级记录
